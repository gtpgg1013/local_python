  - 사람수가 365보다 크면 1
    - 비둘기 집 원리 (확률이 다 똑같으므로 박수 365개 에 점 366개 찍으려고 하면 하나에는 무조건 2개겠지)
  - 사람수가 365보다 작거나 같으면
    - 놀랍게도 23명! - 50.7%
    - 왜?
      - 여집합을 구하변 됨 = 1 - P(다 다를 사건) = 1 - 365Pk/365^k
      - 지수함수 근사법 사용가능
    - P(1팀이라도 같을 확률)
      - 23 - 50.7%
      - 50 - 97%
      - 100 - 99.999%
  - k명이 있다고 할때 중요한건 k명이 아님
    - k명 중 2명 뽑으니 kC2 = k(k-1)/2 가 중요
      - 23명의 경우 253가짓수 : 이제야 직관적으로 23명만 있어도 일어날 수 있겠다 생각듬
